# Royal Ride – Premium Full-Stack Mobility Platform 🛵

**Royal Ride** is a high-performance, full-stack bike rental and subscription platform designed for a seamless, "premium-first" user experience. Developed using **React Native**, **Node.js**, and **MongoDB**, the platform bridges the gap between traditional daily rentals and modern, long-term vehicle subscriptions.

### 🌟 Project Uniqueness
Unlike standard rental tools, Royal Ride implements a dual-business logic:
- **On-Demand Rentals:** Precise daily and weekly booking with real-time availability.
- **Monthly Elite Subscriptions:** Recurring memberships with tiered benefits and tenure-based loyalty discounts.

### 🛠️ Key Technical Highlights
- **Integrated Fintech:** Powered by the **Stripe API**, featuring a native PaymentSheet checkout and a custom built-in **Digital Wallet** for one-tap payments and instant ride extensions.
- **Professional UX/UI:** Adopting a "Royal Yellow" high-contrast theme, the app uses **Reanimated-powered tactile feedback** (`TouchableScale`) and **skeleton loaders** to provide a production-grade feel.
- **Safety & Verification:** A robust **OTP-based authentication** system paired with an optional **KYC verification workflow**, allowing users to upload documents for identity and license approval.
- **Comprehensive Administration:** Includes a companion **Web Admin Dashboard** for monitoring the vehicle fleet, reviewing KYC submissions, managing bookings, and analyzing transaction histories.

### ⚙️ Core Stack
**Mobile:** React Native (TypeScript), NativeWind (Tailwind), Lucide Icons.  
**Backend:** Node.js, Express, JWT, Axios Interceptors (401 safety).  
**Infrastructure:** MongoDB Atlas, Stripe Payments.

This platform is engineered for scalability and production readiness, focusing on visual excellence, secure financial transactions, and intuitive user management.
