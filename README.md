# Royal Ride APP – Premium Full-Stack Mobility Platform 🛵

**Royal Ride** is a high-performance full-stack bike rental and subscription platform designed to deliver a seamless, premium mobility experience. The platform is built using **React Native, Node.js, Express, and MongoDB**, combining real-time booking with subscription-based vehicle access.

---

## 🌟 Project Uniqueness

Unlike traditional rental systems, Royal Ride introduces a **dual business model**:

• **On-Demand Rentals** – Flexible daily and weekly bike bookings with real-time availability.  
• **Monthly Elite Subscriptions** – Recurring memberships offering loyalty rewards, premium bikes, and long-term savings.

This hybrid system bridges the gap between **short-term rentals and modern mobility subscriptions**.

---

## 🚀 Key Features

### Fintech Integration
• Secure payments powered by **Stripe API**  
• Native **PaymentSheet checkout** for seamless transactions  
• Built-in **Digital Wallet** for instant ride extensions and one-tap payments

### Authentication & Security
• **OTP-based authentication** for secure login  
• **JWT-secured REST APIs** for backend communication  
• Optional **KYC verification workflow** for license and identity validation

### Premium User Experience
• Clean **Royal Yellow high-contrast UI theme**  
• **Reanimated-powered interactions** using `TouchableScale`  
• **Skeleton loaders** for smooth loading states

### Admin Dashboard
A companion **web-based admin dashboard** enables:

• Fleet management  
• KYC review and approvals  
• Booking monitoring  
• Transaction history analytics
